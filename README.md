### Cómo usar
Abre `index.html` en cualquier navegador, o sube a Netlify para acceso público.

### Subir a Netlify
1. Sube este repositorio a GitHub
2. En netlify.com → "Import from Git" → selecciona el repo
3. Listo — obtienes un link público

### Versión
